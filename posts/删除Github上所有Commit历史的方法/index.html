删除Github上所有Commit历史的方法

NagaseYami on Dec 122021-12-12T20:26:00+08:00
1 min read

使用Github的时候偶尔会忘记把一些例如SSH或者API密钥的文件加入.gitignore里，导致不小心上传到Github上，这种时候不光要重置密钥，还要把历史Commit删除

步骤

git checkout --orphan latest_branch
git add -A
git commit -am "commit message"
git branch -D master
git branch -m master
git push -f origin master
